Employee Management Application

A simple Employee Management web application built with Spring MVC and MySQL. This project demonstrates a basic CRUD (Create, Read, Update, Delete) workflow for managing employees: adding, editing, and deleting employee records.

Key Features:
Add new employees
Edit employee details
Delete employees
List all employees

Persist data using MySQL

Tech Stack:
Java (JDK 21)
Spring Boot / Spring MVC
Spring Data JPA (or plain JDBC if used)
MySQL
Maven
Thymeleaf (or JSP) for views
